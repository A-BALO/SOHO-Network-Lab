# Small Office Network Lab

## Project Overview
This project demonstrates the creation of a small local area network (LAN) using a network switch, two client PCs, and a web server. The goal was to configure IP addressing, establish device communication, and verify connectivity through network testing.

## Devices Used
- 1 Network Switch
- 2 Client PCs
- 1 Web Server

## Configuration Tasks
- Assigned IPv4 addresses to network devices
- Connected end devices through an Ethernet switch
- Configured the web server for HTTP access
- Tested communication using ping commands

## Testing
Verified successful connectivity between:
- PC1 → PC2
- PC1 → Web Server
- PC2 → Web Server
